ncbreweries {dsbox}R Documentation

Breweries in North Carolina

Description

This dataset is scraped from RateBeer.com on April 18, 2018, and it includes a list of breweries in North Carolina as of that date.

Usage

ncbreweries

Format

A tibble with 251 rows and 7 variables:

name

Name of brewery

city

City where brewery is located

type

Type of brewery (Microbrewery, Client Brewer, Brewpub, Brewpub/Brewery, Commercial Brewery)

beercount

Number of beers brewed at the brewery

est

Year brewery opened

status

Status of brewery (Active or Closed)

url

URL of brewery page on RateBeer.com

Source

Retrieved from https://www.ratebeer.com/breweries/north%20carolina/33/213/ on April 18, 2018.

Examples

ncbreweries
