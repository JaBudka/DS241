fisheries {dsbox}R Documentation

Fishing industry by country

Description

World fisheries harvest for 2005. The tonnage from capture and aquaculture is listed by country. Countries which harvested less than 100,000 tons are not included.

Usage

fisheries

Format

A tibble with 75 rows and 3 variables:

country

Name of country

capture

Tonnage from capture, including fish, crustaceans, molluscs, etc.

aquaculture

Tonnage from aquaculture

Source

https://en.wikipedia.org/wiki/Fishing_industry_by_country

Examples

fisheries
